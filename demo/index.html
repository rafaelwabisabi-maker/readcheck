Why Canvas-Based Measurement?

The core technical innovation that enables ReadCheck to detect violations other tools cannot.

Traditional Tools (axe-core, Lighthouse)

100–1000ms

Call getBoundingClientRect() or getComputedStyle() on each element. Each call forces the browser to recalculate layout (reflow) — blocking the main thread, causing visual jank, scaling linearly with page complexity.

ReadCheck (Canvas via Pretext)

1–5ms

Measure text with CanvasRenderingContext2D.measureText() on an offscreen canvas. Zero reflow, zero paint, zero layout impact. Same font metrics, 500x faster. Scales with text count only, not page complexity.

// The Pretext technique: measure text WITHOUT triggering reflow
const canvas = document.createElement("canvas");
const ctx = canvas.getContext("2d");

// Set canvas font to match the element's computed style
ctx.font = "400 16px Arial";

// Measure — returns width in CSS pixels, sub-pixel accurate
// Cost: ~0.02ms vs ~5ms for getBoundingClientRect()
const width = ctx.measureText(text).width;

// Now we can detect: does this text overflow its container?
// Will it break at 200% zoom? How many chars per visual line?
// These questions are IMPOSSIBLE in DOM-only tools without reflow.

How ReadCheck Works

Layer 0: Canvas Measurement

Uses CanvasRenderingContext2D.measureText() to detect accessibility barriers without triggering DOM reflows. Inspired by Pretext (Cheng Lou), this approach avoids layout thrashing entirely — enabling novel rules that DOM-only tools structurally cannot express.

Detect + Report (No Overlays)

Default mode: detect and report. ReadCheck never injects overlays or modifies your DOM without explicit opt-in. This is a deliberate stance against accessibility overlay products — endorsed by 975 experts at overlayfactsheet.com.

The 96% Coverage Target

The WebAIM Million 2025 analysis finds the same 6 failure categories in 96% of homepages: low contrast, missing alt text, missing labels, empty links, missing lang, and empty buttons. ReadCheck targets all six with measurement-enhanced detection.

🌱

Part of the SilberWelt Initiative

ReadCheck is the technical arm of SilberWelt — an initiative for elderly digital inclusion. While SilberWelt addresses the human side (digital literacy, community support, warm technology), ReadCheck addresses the technical side: making websites readable and usable for older adults through measurement-based accessibility detection. Elderly-specific rules (enhanced contrast thresholds, larger target sizes, cognitive load scoring) are built into the engine.
